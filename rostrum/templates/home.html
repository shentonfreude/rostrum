ROSTRUM

ROSTRUM visualizes data about applications from ROSA and STRAW which have been gardened in a spreadsheet; only current versions are maintained, history is not retained. The spreadsheet is exported as CSV then this application can be used to show all applicaitons as an overview, drill down to the categorized detail of an app, or view categorized attributes based on category of all apps.

How to Use It

Use the menu-bar buttons or the links here. The menubar, like the rest of the site's layout is "responsive", automatically resizing and stacking information based on screen size.

An auto-completing Search box is available on every page, allowing you to quickly find applications by acronym, name, or project manager.

The Overview browses all the applications by acronym and allows drilling down to a specific one to see all the details for them.

The Administrative, Compliance, Description, and Technical views allow you to see details of all apps relevant to these feature categories.

You can load data by exporting ROSA data from a spreadsheet a CSV file, then Uploading the file. You can also export the data from the app and Download the file as a CSV.

The Edit feature allows you to change any of the values for any application. Coupled with the upload and download, this allows "round-tripping" of changes into a spreadsheet.
