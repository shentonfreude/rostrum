ROSA

The Repository of Supported Applications is a web application for browsing and querying information about NASA HQ applications.

Learn more »

How to Use It

You can use the search box in the upper left of each page to search for applications by acronym, name, and description (more attributes later).

Use the menu-bar Acronyms button to browse all the applicaitons by acronym, then drill down to a specific one to see an overview of each release.

The All Versions menu-bar button displays an overview of all applications and all their versions, color-coded by the status of the release.

The menu-bar Search button will eventually offer a sophisticated interface for complex queries, but for now duplicates the quick search box in the upper left of ever page.
